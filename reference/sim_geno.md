# sim_geno

Simulated geno file of eigenstrat format

## Usage

``` r
sim_geno(n_ind, n_snp, filename)
```

## Arguments

- n_ind:

  number of individuals

- n_snp:

  number of SNPs

- filename:

  filename of export

## Value

NULL exports a file

## Examples

``` r
if (FALSE) { # \dontrun{
sim_geno(10, 5, "geno.txt")
} # }
```
